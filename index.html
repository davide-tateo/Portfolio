Natural Language Processing on data scraped from X (Twitter)

The project aims to understand how NLP techniques applied on data scraped from social media can be exploited in business & finance. A Logistic Regression model was used to classify the tweets as Western or Russian. Topic modelling was then performed using LDA and BERTopic. Finally, a sentiment analysis was conducted and the correlation between the sentiments and other variables was investigated.

Cleaning AirBnb Data

This group project involved using a scraped Airbnb dataset focused on Copenhagen listings and hosts in June 2022 to assist MindWonder, a fictional local hospitality start-up. Dividing tasks into data-related steps, we cleaned and analyzed the dataset using Python, and then created insightful visualizations in Tableau. The Hamming Code

For a university assignment, a colleague and I were tasked to create a Python implementation of the Hamming code.
The Hamming code is an error-detecting and correcting code used in digital communication. It adds extra bits to data to create a specific pattern, allowing the detection and correction of single-bit errors. This helps ensure reliable transmission and reception of data in systems like computer memory or data storage.
